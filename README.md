# Netflix Clone

A pixel-perfect Netflix landing page clone built using **HTML, CSS, and JavaScript**.  
This project recreates the layout and design of the Netflix homepage while focusing on responsive design and clean UI structure.

- Responsive design (mobile + desktop)
- Modern UI layout
- Clean CSS structure
- Organized project files
