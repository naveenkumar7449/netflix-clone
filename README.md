# Netflix Clone

A responsive Netflix landing page clone built with HTML, CSS, and JavaScript.

## Features

- **Responsive Navbar** - Netflix logo with language selector and sign-in button
- **Hero Section** - Full-screen banner with email signup form
- **Movie Carousel** - Horizontal scrolling movie list with 10 posters
  - Smooth slide navigation with arrow buttons
  - Animated ranking numbers with glow effect
  - Hover effects on posters (scale + white border)
  - Numbers move upward on hover
- **Feature Cards** - Grid layout showcasing Netflix benefits
- **FAQ Accordion** - Expandable question/answer sections
- **Footer** - Links and contact information

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript

## Installation

1. Clone the repository
```bash
git clone <your-repo-url>
```

2. Open `index.html` in your browser

## Usage

- Click arrow buttons to slide through movie posters
- Hover over posters to see scale and border effects
- Click FAQ questions to expand/collapse answers
- Fully responsive on mobile and desktop
